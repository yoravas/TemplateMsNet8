# Markdown File

La cartella NomeFunzione esiste come esempio di struttura per l'utilizzo di MediatR all'interno del progetto. Puoi rinominare o eliminare questa cartella in base alle tue esigenze specifiche.
Per esempio, se stai implementando una funzionalità per la gestione delle news, potresti rinominare la cartella in "News" e creare i relativi handler, comandi e query all'interno di essa.
