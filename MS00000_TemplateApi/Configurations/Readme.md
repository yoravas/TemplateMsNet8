# Markdown File

In questa cartella vengono inserite le configurazioni necessarie al funzionamento delle sottofunzioni del progetto.
Per esempio: La cartella Serilog con tutti i file di configurazione per il logging; 
La cartella StatusCodesResult con tutti i file di configurazione per i codici di stato custom delle risposte delle API;
La cartella Swagger con tutti i file di configurazione per la documentazione delle API; ecc...
